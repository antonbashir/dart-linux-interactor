# API

### InteractorNotification 

```dart title="Declaration"
class InteractorNotification {
  String getInputString({int? length})
  Pointer<T> getInputObject<T extends Struct>()
  T parseInputObject<T, O extends Struct>(T Function(Pointer<O> object) mapper) 
  void setOutputInt(int data)
  void setOutputBool(bool data)
  void setOutputDouble(double data)
  void setOutputString(String data)
  void setOutputObject<T extends Struct>(void Function(Pointer<T> object) configurator)
  Future<void> setOutputBytes(List<int> bytes) async
}
```

:::note
Remember that `input` and `output` should be allocated and freed by the requester (party which makes a notification).
:::

#### getInputString
Parse the notification into Dart String. You can specify `length` if you need to parse a fixed-length string.

#### getInputObject
Parse the notification into the Dart FFI pointer.

#### parseInputObject
Parse the notification into the Dart object pointer by a custom mapper.

#### setOutputInt
Set the notification output integer value.

#### setOutputBool
Set the notification output bool value.

#### setOutputDouble
Set the notification output double value.

#### setOutputString
Set the notification output string value.

#### setOutputObject
Set the notification output Dart FFI value.

#### setOutputBytes
Set the notification output bytes.
